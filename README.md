# Aprendiendo a usar git

## Configuracion inicial de git
```bash
git config -global user-name <Nombre> #Nombre del usuario
git config -global user-email <email> #Email del usuario
```
## Comandos

```bash
git init #Crea un repositorio
git clone #Clona un repositorio desde el servidor remoto (github)
git add <file> #Agrega archivos al stage
git status #Verifica el estado de los archivos
git commit #Crea una version nueva de los archivos del stage
git log #Muestra todos los commit del proyecto con su respectiva informacion 
git push #sube todos los cambios guardados en local al repositorio remoto
git pull #importa todos los cambios guardados en el repositorio remoto al repositorio local
```
